# 员工异常行为排查系统

🛡️ 基于北京银行员工异常行为清单的完整交互式排查系统

## 📋 项目简介

本系统是基于北京银行《员工异常行为清单_标签与分类》文档开发的交互式网页应用，旨在帮助金融机构进行员工异常行为的识别、分类和排查工作。

### ✨ 核心特性

- **📊 完整数据覆盖**：包含173项异常行为，5大主要分类
- **🔍 智能搜索**：支持关键词搜索和风险等级筛选
- **📱 响应式设计**：适配桌面端和移动端
- **📰 真实案例**：提供银行业真实案例分析和经验教训
- **🎯 精准排查**：详细的个人排查、部门排查和核实方法
- **🚨 风险分级**：高/中/低三级风险等级标识

## 🗂️ 数据分类

### 1. 违法犯罪类 (48项) - 🚨 高风险
- **黄赌毒类** (3项)：嫖娼、赌博、毒品相关活动
- **法律程序及信用类** (3项)：司法程序、失信名单等
- **公共安全秩序类** (5项)：行政拘留、传销、酒驾等
- **经济金融类** (32项)：洗钱、非法集资、挪用资金等
- **数据隐私类** (5项)：泄露客户信息、违规查阅等

### 2. 经济行为类 (30项) - ⚠️ 中风险
- **个人交易异常** (17项)：违规投资、消费异常等
- **个人信贷异常** (8项)：负债过高、贷款逾期等
- **个人账户异常** (4项)：账户关联异常等
- **其他经济行为类** (1项)

### 3. 业务操作类 (83项) - 📋 混合风险
- **违规销售** (12项)：未经批准销售、虚假宣传等
- **违规交易** (6项)：与客户不当交易等
- **违规管理** (13项)：违规决策、滥用职权等
- **违规代理** (2项)：代客保管、代客办理等
- **其他失范行为** (50项)：考勤违规、信息泄露等

### 4. 家庭情况类 (5项) - ℹ️ 低风险
- **家庭矛盾** (1项)：家庭变故造成行为异常
- **家庭亲友风险** (3项)：家属信贷风险、违法活动等
- **其他家庭情况类** (1项)

### 5. 社会交往类 (7项) - ℹ️ 低风险
- **交往不当** (4项)：与企业主关系密切、民间借贷等
- **其他社会交往类** (2项)：同事议论、情况复杂等
- **个人作风不当** (1项)：品德操守问题

## 📦 部署指南

### 方法一：GitHub Pages（推荐）

1. **创建GitHub仓库**
   ```bash
   # 创建新仓库
   git init
   git add .
   git commit -m "初始化员工异常行为排查系统"
   git branch -M main
   git remote add origin https://github.com/你的用户名/仓库名.git
   git push -u origin main
   ```

2. **启用GitHub Pages**
   - 进入仓库设置 (Settings)
   - 找到 Pages 选项
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main"
   - 点击 Save

3. **访问网站**
   - URL：`https://你的用户名.github.io/仓库名`
   - 通常几分钟后生效

### 方法二：本地服务器

```bash
# Python 3
python3 -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js
npx http-server

# 访问 http://localhost:8000
```

### 方法三：Nginx/Apache

将文件复制到 Web 服务器根目录即可。

## 🔧 使用说明

### 基本操作
1. **浏览分类**：点击主页的5个分类卡片
2. **查看子分类**：在分类页面选择具体的子分类
3. **查看行为详情**：点击具体的异常行为
4. **搜索功能**：使用顶部搜索框查找特定行为
5. **风险筛选**：使用筛选按钮按风险等级筛选

### 高级功能
- **面包屑导航**：点击面包屑快速返回上级
- **搜索建议**：未找到结果时查看推荐关键词
- **案例学习**：重点关注真实案例和影响分析
- **排查指导**：参考详细的排查方法指南
